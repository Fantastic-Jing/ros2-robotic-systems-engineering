| 参数名称 | 调整方向 | 带来的影响效果 |
| --- | --- | --- |
| `jda_threshold` | 调大 ($\uparrow$) | **扩大检测距离**。远距离时不容易跟丢目标，但容易把障碍物墙面和目标机器人黏成同一个物体。 |
|  | 调小 ($\downarrow$) | **提升分辨力**。能精确区分挨在一起的物体，但两车距离变远时，目标点云容易断裂导致跟丢。 |
| `width_tolerance` | 调大 ($\uparrow$) | **提高检测成功率**。两车斜对或距离发生急变时不容易丢失目标，但可能误将桌子腿识别为小车。 |
|  | 调小 ($\downarrow$) | **提高目标纯净度**。百分百只过滤真正的 Burger 尺寸，但小车稍微侧向倾斜雷达截面改变时就会丢失。 |
| `kp_linear` | 调大 ($\uparrow$) | **缩短响应延迟**。Leader 加速时 Follower 跟得更紧，但接近 $0.5\text{ m}$ 边界时容易前后刹车顿挫。 |
| `kp_angular` | 调大 ($\uparrow$) | **增强弯道弯曲追踪**。弯道性能好，不容易切弯跟丢，但在直线上小车可能会像喝醉酒一样左右扭摆。 |


| 参数名称 | 调整方向 | 带来的影响效果 |
| --- | --- | --- |
| `critical_brake_dist` | 调大 ($\uparrow$) | **提高安全性**。Follower 会距离障碍物更远时就提前紧急刹车，但可能会在狭窄通道内引发误制动。 |
|  | 调小 ($\downarrow$) | **提高通过性**。允许小车钻过更狭窄的空隙，但如果增益过高可能来不及刹车直接撞墙。 |
| `avoidance_angle_range` | 调大 ($\uparrow$) | **加宽防撞视场**。侧前方有墙壁或障碍物时也能被检测到，但大角度拐弯时容易把斜前方的 Leader 误判为障碍物而错杀。 |
| `max_crumbs` | 调大 ($\uparrow$) | **拉长路径记忆**。能够完整记录更长的复杂弯道轨迹，但会消耗微量内存并增加老数据的滞后时间。 |
